# masaoka_shiki

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This is an open source project for Masaoka Shiki's haiku poems, sourced from the City of Matsuyama, Ehime.

## Features
- Provides a dataset of Masaoka Shiki's haiku poems, organized by season
- Includes tools to download and analyze the haiku data

## Data / API
The haiku data is obtained from the open data portal of the City of Matsuyama, Ehime. The data is licensed under CC-BY.
