# 正岡子規の俳句
松山市がオープンデータとして公開している正岡子規の季節別俳句を収集・整理したリポジトリです。

## デモ
[俳句データの表示ページ](https://code4fukui.github.io/masaoka_shiki/)

## 機能
- 春・夏・秋・冬・新春・雑の6つの季節別に俳句を収集
- 重複する俳句を特定
- 収集した俳句データをCSV/JSON/CBORで提供

## 必要環境
特に必要ありません。

## 使い方
各種データファイルをダウンロードして利用してください。

## データ・API
このリポジトリで公開しているデータは、[松山市オープンデータ](https://www.city.matsuyama.ehime.jp/shisei/opendata/metadata/shikihaiku.html)から取得したものです。
